# Chess piece images — attribution

The PNG chess piece images in this directory (`*.png`) are rasterized from
the **"SVG chess pieces"** set created by **Colin M.L. Burnett** (Wikimedia
Commons user [*Cburnett*](https://commons.wikimedia.org/wiki/User:Cburnett)).

- **Source:** https://commons.wikimedia.org/wiki/Category:SVG_chess_pieces
- **Original format:** SVG. **Modification:** converted to PNG and renamed to
  the short `<piece><color>t.png` form used by this widget.

The originals are multi-licensed by the author under the GNU Free
Documentation License, Creative Commons Attribution-ShareAlike 3.0, the BSD
3-Clause License, and the GNU General Public License v2+. **We use them under
the BSD 3-Clause License** (reproduced below), which is compatible with this
project's MIT license.

## File → original source mapping

Each file derives from `Chess_<name>45.svg` on Wikimedia Commons, where the
name encodes piece + color + transparent background:

| File      | Piece  | Color | Original SVG          |
|-----------|--------|-------|-----------------------|
| `klt.png` | King   | White | `Chess_klt45.svg`     |
| `kdt.png` | King   | Black | `Chess_kdt45.svg`     |
| `qlt.png` | Queen  | White | `Chess_qlt45.svg`     |
| `qdt.png` | Queen  | Black | `Chess_qdt45.svg`     |
| `rlt.png` | Rook   | White | `Chess_rlt45.svg`     |
| `rdt.png` | Rook   | Black | `Chess_rdt45.svg`     |
| `blt.png` | Bishop | White | `Chess_blt45.svg`     |
| `bdt.png` | Bishop | Black | `Chess_bdt45.svg`     |
| `nlt.png` | Knight | White | `Chess_nlt45.svg`     |
| `ndt.png` | Knight | Black | `Chess_ndt45.svg`     |
| `plt.png` | Pawn   | White | `Chess_plt45.svg`     |
| `pdt.png` | Pawn   | Black | `Chess_pdt45.svg`     |
